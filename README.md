## ðŸ§  What is Physical AI?

Physical AI refers to AI systems that:
- Interact with the **physical world**
- Perceive through sensors
- Act through motors or actuators
- Learn under real-world constraints (noise, latency, physics)

Examples include:
- Robots
- Autonomous vehicles
- Drones
- Smart physical agents
